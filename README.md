# Salon Booking API

A simple REST API built with Express.js.

## Endpoints

GET /bookings
Returns all bookings.

GET /bookings/:id
Returns one booking.

POST /bookings
Creates a booking.

PUT /bookings/:id
Updates a booking.

DELETE /bookings/:id
Deletes a booking.

## Run

npm install

npm run dev

Server runs on port 3000.
